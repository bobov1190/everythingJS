JavaScript #1: объектная модель документа DOM и BOM


Javascript может запускаться везде , где есть Движок JavaScript

С помощью Браузера Js может управлять содержимым html документа

Объектная модель документа (Document Object Model, DOM)
- для управления содержимым HTML-документа

Объектная модель Браузера (Browser Object Model, DOM)
- дополнительные объекты , предоставляемые Браузером


window - DOM - Document
       - BOM - navigator
       - JavaScript - Object

Чтобы увидеть дерево DOM в браузере, нужно открыть вкладку «Elements».

По идее директива <!DOCTYPE html> тоже помещается в это дерево как самый верхний узел,
но мы ее не будем изображать для экономии пространства и времени.

Формально дерево DOM содержит 12 различных типов узлов. Но мы, в основном, будем использовать такие:

document – объект, представляющий дерево DOM (точка входа в DOM)
узлы-элементы – теги HTML-документа
текстовые узлы – элементы, содержащие текстовую информацию
комментарии – элементы с комментариями (иногда они используются JS для записи и чтения информации, скрытой от пользователя)
